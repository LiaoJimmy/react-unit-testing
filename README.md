# react-unit-testing

React.js unit testing tutorial

# Run the project

1. Install the dependencies

```bash
npm install
```

2. Run the storybook

```bash
npm run storybook
```

3. Run the test with watch mode

```bash
npm run test:watch
```

4. Run the test with coverage

```bash
npm run test
```
